Aula 18 — SQL Injection (Vulnerável) vs Prepared Statements (Seguro)

⚠️ Sistema VULNERÁVEL

🧪 Ataque SQL Injection: Usuário: ' OR '1'='1' --
Senha: abc
⚙️ SQL gerado pela aplicação:
// ❌ Concatenação direta — VULNERÁVEL
$sql = "SELECT * FROM usuarios
  WHERE usuario = '$usuario'
  AND senha = '$senha'";
$conn->query($sql);

✅ Sistema SEGURO

✅ Tente o mesmo ataque: Usuário: ' OR '1'='1' --
Será bloqueado! Login válido: admin / senha123
⚙️ Prepared Statement gerado:
// ✅ Prepared Statement — SEGURO
$stmt = $conn->prepare(
  "SELECT * FROM usuarios
   WHERE usuario=? AND senha=?"
);
$stmt->bind_param("ss", $usuario, $senha);
$stmt->execute();
